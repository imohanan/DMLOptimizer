***Stats scripts are for Linux OS***

Steps to be performed once during project initialization
0) Install python, matplotlib library
1) Go to the folder where scripts are stored using terminal.
2)Run: chmod +x stats.sh
3)Run: chmod +x stopstats.sh
4)Run: chmod +x value_extractor.sh

Steps to be performed during each run
1) Run ./stats.sh just before running java program in eclipse
2) Run ./stopstats.sh just after the java program finishes.
3) Copy the graph plots to your desired location for comparison.
4)You may want to delete or store the other files as per user's wish(these files have cpu, disk, memory and network traces)
